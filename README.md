# 🎓 PCM Smart Study AI

AI-powered study assistant for Physics, Chemistry, and Mathematics built using Python and Streamlit.

This project helps students solve questions, understand concepts, generate practice tests, and create AI notes — all in one place.

---

## 🚀 Features

- Solve Questions Instantly  
- Solve Questions from Images  
- Practice Question Generator  
- Concept Explainer  
- Formula Generator  
- Practice Test Mode  
- AI Notes Generator  
- AI Quiz Mode  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- AI API  
- Prompt Engineering

## ⚙️ Installation

### 1. Clone Repository

git clone https://github.com/kalpeshp5113/pcm_ai_helper.git

### 2. Go to Folder

cd pcm_ai_helper

### 3. Install Requirements

pip install -r requirements.txt

### 4. Run App

streamlit run app.py

---

## 🔐 Add API Key

Create .env file and add:

API_KEY=your_api_key_here

---

## 💡 How It Works

1. Select study mode  
2. Enter question or upload image  
3. AI processes input  
4. Get instant explanation
